Just a vi setup that works for me.
I can create multiple tabs in GNOME Terminal (CTRL+SHIFT+T) and switch between them using CTRL+PgDn and CTRL+PgUp.
Additionally, I can navigate through the files with the NERDTree plugin and the tags generated by the ctags command.
I'm sure it's not the best setup ever, but it's something that works for me, and I want to back it up here so I don't lose it.
